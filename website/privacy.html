隐私政策

最后更新：2026年3月16日

简介

简历投递助手（以下简称"本扩展"）是一款 Chrome 浏览器扩展程序，旨在帮助用户在招聘网站上自动填写简历信息。我们非常重视您的隐私保护。

数据收集

本扩展不会收集、传输或存储任何用户个人数据到远程服务器。具体说明如下：

  • 您的简历数据仅存储在浏览器本地（Chrome Storage），不会上传到任何外部服务器。
  • 本扩展不使用任何第三方分析或跟踪工具。
  • 本扩展不收集浏览历史、Cookies 或任何用户行为数据。

AI 功能说明

当您主动使用 AI 辅助填写功能时，页面的表单元数据（字段名称、标签等结构信息）可能会发送至您自行配置的 AI API 进行分析。我们不会发送您的个人简历内容，仅发送表单结构信息。

数据存储

所有简历数据使用 Chrome 的 chrome.storage.local API 存储在您的本地浏览器中。您可以随时在扩展的设置页面中查看、修改或删除这些数据。

权限说明

  • activeTab：用于在当前标签页中扫描和填写表单。
  • storage：用于在本地存储您的简历数据和设置。
  • scripting：用于在招聘网站页面中注入表单填写脚本。
  • clipboardWrite/Read：用于通过剪贴板方式填写某些受保护的表单字段。
  • host_permissions (all_urls)：用于在各招聘网站上自动填写表单。

数据安全

由于所有数据仅存储在您的本地设备上，数据安全取决于您设备本身的安全性。我们建议您定期备份简历数据。

第三方服务

本扩展不依赖任何第三方服务来处理用户数据。唯一可选的第三方交互是用户自行配置的 AI API 接口。

未成年人

本扩展主要面向求职者使用，不针对13岁以下未成年人。

政策更新

我们可能会不时更新本隐私政策。任何重大更改将通过扩展更新通知用户。

联系方式

如有隐私相关问题，请通过 GitHub Issues 联系我们。
