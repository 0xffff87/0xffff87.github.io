常见问题

插件支持哪些招聘网站？

目前支持百度招聘（talent.baidu.com）、腾讯招聘（careers.tencent.com）、字节跳动招聘等主流平台，并且 AI 兜底机制可以适配大部分标准招聘表单。

我的个人数据安全吗？

完全安全。所有数据仅存储在你的浏览器本地（localStorage），不会上传到任何服务器。导出的 JSON 文件也仅保存在你的本地设备上。

如何将网页填写的数据同步到插件？

在网页中填写完信息后，点击「导出 JSON」按钮下载数据文件。然后打开 Chrome 插件弹窗，点击底部的「导入」按钮，选择刚下载的 JSON 文件即可。

填写后表单数据不正确怎么办？

如果自动填写结果不准确，你可以手动修正。插件会持续优化匹配规则，你也可以在 GitHub 上反馈问题。
