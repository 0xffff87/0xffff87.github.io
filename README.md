# 简历投递助手 - Chrome 浏览器插件

AI 驱动的简历投递助手，帮助你在各大招聘网站和公司招聘官网上智能填写简历表单。

## 功能特点

- **个人信息管理**：在插件中预先填写姓名、性别、出生日期、联系方式、教育经历、实习/工作经历、获奖信息、技能等
- **规则匹配 + AI 兜底**：优先使用本地正则规则匹配字段（零延迟），规则未覆盖的字段通过 AI 大模型智能识别
- **多种控件支持**：支持文本输入框、原生下拉框、自定义下拉组件（brick-select、ant-select、el-select、ud__select）、日期选择器、单选/多选按钮等
- **多 AI 服务商**：支持 SiliconFlow、DeepSeek、智谱 AI、Moonshot（Kimi）、OpenAI 以及自定义 API
- **一键填写**：在任意招聘页面点击悬浮按钮或使用快捷键 `Ctrl+Shift+F`
- **数据导入导出**：支持将简历数据导出为 JSON 文件备份，或从文件导入恢复
- **可拖拽悬浮按钮**：右下角悬浮按钮可自由拖拽，避免遮挡页面内容

## 安装方法

1. 下载或克隆本项目到本地
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本项目的根目录

## 使用步骤

### 第一步：配置 AI 服务

1. 安装插件后会自动打开设置页面（也可以点击插件图标右上角的齿轮图标进入）
2. 在「AI 服务配置」区域选择服务商（推荐 SiliconFlow，支持多模态且性价比高）
3. 填写 API Key（需要到对应服务商官网注册获取）
4. 点击「测试 AI」确认配置正确
5. 点击「保存设置」

### 第二步：填写个人信息

1. 点击浏览器工具栏中的插件图标
2. 在弹出的界面中依次填写：
   - **基本信息**：姓名、性别、出生日期、手机号、邮箱、民族、政治面貌、籍贯等
   - **教育经历**：学校、学历、专业、起止时间、GPA 等
   - **实习/工作经历**：公司、职位、起止时间、工作描述等
   - **项目经历**：项目名称、角色、技术栈、描述等
   - **技能/其他**：获奖、专业技能、证书、语言能力等
3. 点击「保存信息」
4. 可点击底部的「导出」按钮备份数据

### 第三步：一键填写

1. 打开任意招聘网站的简历填写页面
2. 使用以下任一方式触发填写：
   - 点击页面右下角的悬浮按钮 → 点击「开始填写」
   - 在插件弹窗中点击「一键填写」
   - 使用快捷键 `Ctrl+Shift+F`
3. 插件会自动：
   - 扫描页面上的所有表单字段（含自定义 UI 组件）
   - 通过规则引擎进行本地匹配（毫秒级）
   - 对规则未覆盖的字段，截图+元数据发送给 AI 分析
   - 逐个自动填充匹配到的信息
4. 填写完成后请检查填充结果，根据需要手动调整

## 项目结构

```
├── manifest.json          # Chrome MV3 扩展清单
├── background/
│   └── background.js      # 后台服务（AI API 调用、截图、快捷键）
├── popup/
│   ├── popup.html          # 弹窗界面
│   ├── popup.css           # 弹窗样式
│   └── popup.js            # 弹窗逻辑（简历录入、导入导出）
├── content/
│   ├── content.js          # 内容脚本（表单扫描、规则匹配、字段填充）
│   └── content.css         # 悬浮面板与按钮样式
├── options/
│   ├── options.html        # 设置页面
│   ├── options.css         # 设置样式
│   └── options.js          # 设置逻辑（后端+AI 配置）
├── icons/                  # 插件图标（16/48/128px）
├── server/

├── test/                   # 自动化测试脚本
└── README.md               # 本文件
```

## 注意事项

- 你的个人信息和 API Key 仅保存在浏览器本地存储中，不会上传到任何第三方服务器
- AI 填写的结果可能不完全准确，请务必在提交前人工检查
- 不同招聘网站的表单结构差异较大，部分复杂控件可能需要手动填写
- 推荐使用支持多模态（图片识别）的 AI 模型，可以结合截图更准确地识别表单

## 隐私说明

- 所有数据均存储在本地 `chrome.storage.local` 中
- AI API 调用直接从你的浏览器发出，插件本身不设中转服务器
- 页面表单信息仅在你主动点击"填写"时才会发送给 AI 进行分析
- 后端服务（server/app.py）为可选组件，仅用于授权和日志收集
